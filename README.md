# FIGLE_M

## 개발 목적
피파온라인4 구단명 검색으로 전적 및 선수 플레이 정보를 제공합니다.
심플한 UI로 누구나 쉽게 접근할 수 있도록 제작하였습니다.

1 ON 1 전적검색을 제공하며, 추가 기능에 대해서 업데이트 예정입니다.
